gitBook 安装与运行

前提条件

  • Node.js 环境
  • mac os

    开始安装

    • 全局安装gitBook
      
sudo npm install gitbook-cli -g
    • 检查版本
      
gitbook --version
CLI version: 2.3.2
Installing GitBook 3.2.3
    • 创建目录 并 初始化gitBook 项目
      
mkdir myBook
cd myBook
gitbook init
    • 在根目录运行
      
gitBook server
    • 访问 http://localhost:4000 就出来啦
